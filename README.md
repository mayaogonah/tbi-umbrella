R/Quarto repository for the manuscript 'An umbrella review of health outcomes following traumatic brain injury' published in Nature Mental Health.

Journal version of the manuscript is [here](https://www.nature.com/articles/s44220-024-00356-5).

The reposity contains code to convert effect sizes to relative risk, compare risk estimates of health outcomes following traumatic brain injury on a forest plot, and calculate the population attributable fraction. [The HTML file](https://mayaogonah.github.io/tbi-umbrella/) rendered from Quarto, outlines the key stages of analysis.
